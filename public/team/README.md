## How to Add Images

1. Place the Studio Ghibli-style portraits in this directory
2. Update the About page at `src/app/(marketing)/about/page.tsx`
3. Replace the `img: null` with the image path:
   ```tsx
   img: '/team/richard-bui.png'
   ```
4. The images will automatically display in the circular frames

## Image Specifications

- **Format**: PNG or JPG
- **Recommended Size**: 800x800px (square)
- **Style**: Studio Ghibli anime aesthetic with warm, dreamy colors
- **Background**: Soft gradients with nature elements (clouds, sky, foliage)
